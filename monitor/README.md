# 无级移动智能监控系统 / Monitor System

端到端能力，涵盖全域无死角移动监控、设备巡检、状态感知及异常预警。

## 模块结构

```
monitor/
├── hardware/    # 硬件设计（云台、移动平台等）
├── firmware/    # 嵌入式固件
├── software/    # 上位机软件与算法
└── docs/        # 模块文档
```
