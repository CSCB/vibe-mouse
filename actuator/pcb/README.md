# 线路板 / PCB

执行器驱动与控制电路板设计。

## 规划中的板卡

| 板卡 | 功能 | 状态 |
|---|---|---|
| 驱动板 | 电机驱动、电流环控制 | 待补充 |
| 电源板 | 电源输入与保护 | 待补充 |
| 编码器板 | 位置反馈读取 | 待补充 |

## 目录结构

```
pcb/
├── driver_board/     # 驱动板
├── power_board/      # 电源板
└── encoder_board/    # 编码器板
```

每个板卡目录下包含：
- 原理图（Schematics）
- PCB 布局
- BOM 清单
- Gerber 生产文件
