## 组装流程

1. **查阅 BOM** — 准备非打印标准件（磁铁、轴承、螺丝等）
2. **3D 打印** — 打印所有部件模型
3. **绕线** — 参考定子绕线图进行定子绕线
4. **组装** — 参考总装配图 STEP 文件进行组装
